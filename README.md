# pixel-coordinate-regression

Pixel Coordinate Prediction using Deep Learning

Problem:
Predict the (x, y) location of a bright pixel in a 50x50 grayscale image.

Approach:
- Synthetic dataset generation
- CNN model
- Regression output for coordinates

How to Run:
1. Install requirements:
   pip install -r requirements.txt
2. Open the notebook
3. Run all cells
